## 🎯OBJECTIVES

The objective of this project is to develop a simple and efficient application to manage daily expenses. The system allows users to add, view, search, delete, and analyze expenses without using external files.

## 🧩MODULES 
Login Module
Allows user to create and verify password
Add Expense Module
Takes date, category, and amount from user
View Expense Module
Displays all stored expenses
Search Module
Searches expenses based on date
Monthly Report Module
Calculates total expenses for a given month
Total Expense Module
Shows overall spending
Delete Module
Deletes a selected expense
## ⚙️REQUIREMENTS
Software Requirements
C++ Compiler (CodeBlocks / VS Code / Turbo C++)
Basic C++ Libraries (iostream, vector, string)
Hardware Requirements
Computer or Laptop
Minimum 4GB RAM
## 🔬METHODOLOGY
The project is implemented using C++ programming language
Data is stored using vector (dynamic array)
User input is handled using getline() to avoid errors
Operations like add, delete, search are performed on vector
Menu-driven approach is used for user interaction
## ✨FEATURES
User-friendly menu system
Input validation
Dynamic data storage
No file dependency
Fast execution
Simple and efficient design
## ⚠️LIMITIATIONS
Data is not saved permanently
Data is lost after program exit
No graphical interface
